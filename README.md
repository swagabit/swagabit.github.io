# Сайт-визитка · Кирилл Таранов

Статичный лендинг без сборки: открой `index.html` или задеплой папку как есть.

## Как заменить заглушку на видео

1. Кинь файл в `media/videos/`, например `showcase-1.mp4`.
2. Открой `js/data.js`, найди запись с нужным `id` и поменяй:
   `src: null` → `src: "media/videos/showcase-1.mp4"`.
3. По желанию: превью-картинка в `media/posters/showcase-1.jpg`, поле `poster`.

Где что: `SHOWREEL` (большое видео в шапке), `CASES` (кейсы), `SHOWCASE` (сетка роликов).
Просмотры/подписи роликов тоже правятся в `data.js` (поля `tag`, `metric`).

**Формат видео:** MP4 (H.264), вертикальное 720×1280, без звуковой дорожки, до ~6 МБ.
Команда: `ffmpeg -i вход.mp4 -vf scale=720:1280 -crf 28 -an выход.mp4`

## Контакты (обязательно заполнить)

В `js/data.js`, блок `CONTACTS`:
- WhatsApp: вставь номер вместо нулей (формат `9715XXXXXXXX`, без плюса)
- Telegram: вставь юзернейм вместо `username`

## Фото «Обо мне» и OG-картинка

- Фото: `media/photos/about.jpg` (пока стоит заглушка в HTML, скажи Клоду вставить)
- Превью ссылки в мессенджерах: `media/photos/og-image.jpg` (1200×630)

## Тексты

- Статичные тексты (оба языка): `js/i18n.js`
- Кейсы и ролики: `js/data.js`

## Локальный просмотр

`python3 -m http.server 8000` в папке сайта → http://localhost:8000
(просто открыть index.html двойным кликом не выйдет: модули JS требуют сервер)
